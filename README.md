# Power BI Operations Dashboard

A multi-source Power BI dashboard consolidating revenue, vendor operations, and project performance data — designed for real-time C-suite decision-making.

## 📊 Dashboard Pages

| Page | Description |
|---|---|
| `Executive Summary` | High-level KPIs and trend lines for leadership |
| `Revenue Analytics` | Revenue breakdown by segment, region, and period |
| `Vendor Operations` | Vendor spend, payment performance, and contract status |
| `Project Performance` | Project delivery, milestone tracking, and resource utilization |

## 🔍 Key Features

- Real-time data refresh via automated pipelines
- Dynamic slicers for period, department, and region
- Drill-through capability from summary to transaction level
- RAG status indicators (Red / Amber / Green) for instant insight
- Mobile-optimised layout for on-the-go access

## 🛠️ Tools & Stack

| Layer | Tool |
|---|---|
| **Data Sources** | Excel, SQL Server, CSV feeds |
| **Transformation** | Power Query (M Language) |
| **Modelling** | DAX measures and calculated columns |
| **Visualisation** | Power BI Desktop & Service |
| **Automation** | Scheduled refresh + dataflow pipelines |
